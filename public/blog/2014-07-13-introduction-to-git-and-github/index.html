Introduction to git and GitHub

git安装

推荐 Homebrew用于管理常用开源软件包在Mac OS X上的安装和升级。Homebrew用Ruby语言开发，支持千余种开源软件在 Mac OS X 中的部署和管理。Homebrew 项目托管在 Github上。

  1. To install Homebrew, run following command in terminal. Note that ruby is installed by default on Mac OS X.
$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  1. It is quite easy to install git through Homebrew.
$ brew install git

git 常用命令

  1. 将github上面的项目克隆到本地
$ git clone git@github.com:caijun/blog.tonytsai.name.git
  1. 查看当前项目下所有文件的状态
$ git status

状态（包括新建、删除、修改）发生改变的文件用红色标出。

  1. 将更新后的项目提交到git本地仓库
$ git add .

.表示当前项目目录下的所有内容。
若更新包括删除文件，该命令在git 2.0以上版本失效，并产生warning提示使用git add --all代替。

  1. 对更新内容确认并添加确认信息
$ git commit -m 'commit message'

确认信息对于版本控制很关键，每次版本更新时都应该添加描述准确的确认信息。

  1. 将项目提到github，首先要与github上面对应的远程仓库建立连接
$ git remote add origin git@github.com:caijun/blog.tonytsai.name.git
  1. 查看当前项目远程连接的仓库地址
$ git remote -v
  1. 将本地的项目提交到远程仓库
$ git push -u origin master
